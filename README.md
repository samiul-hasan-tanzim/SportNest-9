# SportNest (Client)

## Purpose
SportNest is a modern, high-performance sports facility booking application built with Next.js. It aims to bridge the gap between sports enthusiasts and facility owners by providing a seamless platform to discover, book, and manage sports venues like football fields, cricket nets, tennis courts, and more.

## Live URL
[https://sport-nest-9.vercel.app](https://sport-nest-9.vercel.app)

## Features
- **Comprehensive Dashboard:** View all available facilities, manage your bookings, and track your sports activities.
- **Facility Management:** Owners can easily add, update, and manage their sports facilities through a dedicated interface.
- **Advanced Search & Filtering:** Search for facilities by name or filter them by category (e.g., Football, Cricket, Tennis, Badminton).
- **Secure Authentication:** Robust user authentication system supporting Google Login and Email/Password, powered by Better Auth.
- **Modern & Responsive UI:** A sleek, mobile-first design built with Tailwind CSS and HeroUI components.
- **Dark & Light Mode:** Seamlessly switch between dark and light themes for an optimized viewing experience.
- **Interactive Feedback:** Real-time toast notifications for user actions such as successful bookings, login, and updates.
